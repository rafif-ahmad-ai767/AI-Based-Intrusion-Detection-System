# AI-Based Intrusion Detection System (IDS)

This project implements an intelligent Intrusion Detection System using
Machine Learning to detect cyber attacks such as DDoS, Port Scanning,
and Brute Force. The model is trained on the CICIDS2017 dataset and
includes a real-time dashboard for monitoring and alerting.

## Features
- ML-based anomaly detection
- Supports multiple attack types
- Real-time dashboard (Streamlit)
- High accuracy Random Forest model

## Tech Stack
Python, Scikit-Learn, Pandas, Streamlit

## Dataset
CICIDS2017 – Canadian Institute for Cybersecurity
